# TypeScript Custom Resolver Plugin - Инструкция по редактированию

## 1. Перейти в директорию плагина в терминале
```bash
cd custom-scripts/typescript-plugin
```

## 2. Установить зависимости
```bash
npm i
```

## 3. Редактировать код плагина
Внесите необходимые изменения в файлы плагина.

## 4. Собрать плагин
```bash
npm run build
```

## 5. Обновить основной проект
Вернитесь в корень проекта и обновите зависимости:
```bash
cd ../..
npm i
```

## 6. Перезагрузить TypeScript Server
В VS Code:
- Нажмите `Ctrl+Shift+P`
- Выберите `Developer: Reload Window`

Или, если доступна такая опция:
- Нажмите `Ctrl+Shift+P`
- Выберите `TypeScript: Restart TS Server`

*Примечание: В VS Code функция "Go To Definition" может начать работать некорректно на активной вкладке после перезагрузки с новой конфигурацией TS-сервера. Закройте и откройте заново (`Ctrl+Shift+T`) активную вкладку в случае некорректного поведения*
---

**Важно**: После каждого изменения плагина необходимо выполнить все шаги, иначе изменения не вступят в силу.

**Важно**: Папка `dist` в папке плагина должна оставаться в индексации git (быть в исключениях через `!` в `.gitignore`), чтобы плагин можно было установить в зависимости проекта
